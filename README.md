### 关键特性

* 2x 150 MHz Cortex-M33/Hazard3
* 520KB SRAM + 16MB QSPI Flash + 8KB OTP
◦ 2× UARTs
* 2× SPI controllers
* 2× I2C controllers
* 24× PWM channels
* USB 1.1 controller and PHY, with host and device support
* 12× PIO state machines
* 1× HSTX peripheral
* 封装：QFN-80 (10×10 mm) and QFN-60 (7×7 mm)
